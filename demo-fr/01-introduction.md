
# 1. Introduction

## Le scénario de risque Contoso &amp; pourquoi il est important

- Contoso, une entreprise mondiale de services financiers, possède des dizaines de PDF internes portant sur des sujets comme le risque financier, la cybercriminalité, le blanchiment d’argent et l’instabilité politique.
- Ces documents sont stockés dans SharePoint et représentent un savoir propriétaire essentiel pour les gestionnaires de risques et les équipes d’investissement.
- Défi :
  - Comment rendre ces données non structurées facilement consultables et exploitables ?
  - Elles doivent rester sécurisées et accessibles uniquement aux utilisateurs de l’entreprise.
- Solution :
  - Combiner les services Microsoft Azure et les capacités de recherche avancées d’Elastic pour créer un assistant alimenté par l’IA, intégré à Microsoft Teams.
